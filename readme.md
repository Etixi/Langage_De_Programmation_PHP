# Windows Installation

+ [php](https://www.php.net/downloads.php)

# Choisir un editeur de texte : Atom

```
choco install atom
```

# Windows CLI

```
echo %PATH%
php -v
```

# Lancer le Server Sur votre CMD

```
php -S localhost:4000
```
